- License / terms of use: Refer to the Kaggle dataset page for the current license and usage terms.

## Project Outputs

The generated MRI outputs in this project are derived from experiments using the dataset above. The local output folders include results for models such as DCGAN, cDCGAN, WGAN, WGAN-GP, and conditional WGAN-GP.
