# MRI GAN

This repository contains notebooks, code, and selected result figures for MRI image generation experiments using GAN variants, including DCGAN, cDCGAN, WGAN, WGAN-GP, and conditional WGAN-GP models.

## Dataset

The experiments use the public Kaggle dataset:

- Dataset: Brain Tumor MRI Dataset
- Source: https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
- Author / uploader: Masoud Nickparvar
- Access date: 2026-07-07
- License / terms: see the Kaggle dataset page for the current license and usage terms

More details are provided in [DATASET.md](DATASET.md).
